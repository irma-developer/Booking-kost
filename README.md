# ğŸ  Web Booking Kost

Web Booking Kost adalah aplikasi berbasis web yang dirancang untuk memudahkan proses pencarian, pemesanan, dan pengelolaan kost secara online. Aplikasi ini membantu calon penyewa menemukan kost sesuai kebutuhan, sekaligus membantu pemilik kost dalam mengelola data kost, kamar, dan pemesanan secara lebih terstruktur dan efisien.

Aplikasi ini dibangun dengan fokus pada kemudahan penggunaan, tampilan yang sederhana, serta alur pemesanan yang jelas agar dapat digunakan oleh berbagai kalangan.

---

## ğŸ¯ Tujuan Aplikasi
- Memudahkan pengguna dalam mencari dan memesan kost secara online  
- Membantu pemilik kost mengelola data kost dan kamar dengan lebih rapi  
- Mengurangi proses manual dalam pencatatan pemesanan  
- Menyediakan informasi kost yang jelas, lengkap, dan mudah diakses
